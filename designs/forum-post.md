## *Fleeca Now*: A Lightweight Money Transfer App for FiveM Servers
A clean and intuitive money transfer application, inspired by **Venmo**, designed to provide a new way for players to handle in-game financial transactions.

![pciture%](upload://veuCeI8gDAQkJeeJS4vZykzF9aS.jpeg)

This resource works out of the box with the following frameworks:
* Ox
* ESX
* Qbox

---

Fleeca Now integrates with LB Phone, allowing players to securely send and receive money from each other. This eliminates the need for players to share personal phone numbers or bank account details for simple transactions.

Buy [🔗 on tebex](https://maximus-scripts.tebex.io/package/6947140) (18 EUR - excl. VAT).

Technical details:
* The resource is built with modern web technologies (TypeScript, React) for stability.
* The main logic and UI are provided as minified, transpiled files. Editable bridge files are included for easy framework and logging integration.

> **ℹ️ Note:** 
> Support is provided for the original, unmodified files. To ensure a stable and bug-free experience, modifications to the core, transpiled files are not supported.
